# 再漫画签到

基于 [qd-today/templates](https://github.com/qd-today/templates) 仓库中的 `再漫画.har` 规则改写为 Python 脚本。  
保留基础的签到功能。使用青龙统一的通知接口进行通知推送。

> 来源说明：该功能流程参考 `再漫画.har`（登录、签到、信息查询接口顺序与字段提取规则）。
> 合规说明：上游仓库未提供明确的开源许可证文件（LICENSE）；请按其 README 声明仅用于学习交流、勿用于商业用途。

## 使用方式

- 确保青龙的依赖管理 - Python3 中安装了 `requests`
- 创建环境变量 `ZAIMANHUA_DEBUG`，值填数字 0
  - 调试时修改为数字1即可开启调试模式输出更多信息。
- 创建环境变量`ZAIMANHUA_ACCOUNTS`，填写再漫画的账号密码。

### 账号密码填写格式如下：
```
[账号][井号][密码][井号][备注]
admin#password#测试
```
