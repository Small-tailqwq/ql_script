# 酷街区签到

基于 [mxyooR/Kuro-autosignin](https://github.com/mxyooR/Kuro-autosignin) 精简而成，遵循 GPL-3.0 协议。  
保留基础的签到、日活功能。使用青龙统一的通知接口进行通知推送。

## 使用方式

- 确保青龙的依赖管理 - Python3 中安装了 `requests`
- 首先通过 [mxyooR/Kuro_login](https://github.com/mxyooR/Kuro_login) 获取到酷街区的 token 信息。
- 在青龙面板中创建环境变量 `KURO_TOKEN` 填入 token 即可。

### token填写格式如下：
```
[token本体][井号][备注]
eyJhbGciO******************.eyJjcm************************************.***************************************#自己号
eyJhbGciO******************.eyJjcm************************************.***************************************#小号
```
