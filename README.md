# ql_script
自用青龙脚本，基于开源项目魔改的单文件版，使用环境变量存储身份信息，调用青龙统一的消息推送进行管理。

本项目基于多个其他开源项目构建而成，各个脚本的许可受各自原本的许可证限制。

- `skyland/` 目录遵循 MIT，保留原作者版权声明并追加修改署名。
